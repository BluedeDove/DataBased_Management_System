# 项目使用指南

## 📌 项目概述

智能图书管理系统是一个完整的企业级图书馆管理解决方案，实现了图书、读者、借阅的全流程管理。

## 🎯 核心功能模块

### 1. 登录系统
- 支持管理员和图书管理员两种角色
- 默认账号: admin / admin123
- 支持修改密码功能

### 2. 仪表盘
- 实时显示系统统计数据
  - 图书总数和可借数量
  - 读者总数和活跃读者
  - 在借图书和逾期情况
- 图书类别分布饼图
- 热门图书TOP 10
- 活跃读者TOP 10

### 3. 图书管理
- **查询功能**
  - 关键词搜索（书名、作者、ISBN）
  - 按类别筛选
  - 查看图书详情和借阅状态

- **图书操作**
  - 新增图书（填写ISBN、书名、作者等信息）
  - 编辑图书信息
  - 增加馆藏（同一本书增加复本）
  - 图书状态管理（正常、损坏、丢失、销毁）

- **类别管理**
  - 创建图书分类
  - 管理类别关键词

### 4. 读者管理
- **读者信息**
  - 新增读者（读者编号、姓名、类别等）
  - 编辑读者信息
  - 按姓名、编号、电话搜索

- **读者证管理**
  - 读者证激活
  - 读者证挂失
  - 读者证续期
  - 查看读者借阅统计

- **读者种类**
  - 学生类：最多借5本，借期30天
  - 教师类：最多借10本，借期60天
  - 职工类：最多借8本，借期45天
  - 可自定义新的读者种类

### 5. 借还管理
- **借书流程**
  1. 输入读者编号（可扫描）
  2. 输入图书ISBN（可扫描）
  3. 系统自动验证：
     - 读者证是否有效
     - 是否达到借阅上限
     - 是否有逾期图书
     - 图书是否可借
  4. 确认借出，自动计算归还日期

- **还书流程**
  1. 搜索读者
  2. 查看该读者的借阅记录
  3. 选择要归还的图书
  4. 系统自动计算罚款（如逾期）
  5. 确认归还

- **续借功能**
  - 最多续借2次
  - 每次延长一个借期
  - 逾期图书不能续借

- **借阅记录**
  - 查看所有借阅历史
  - 按读者或图书查询
  - 查看借阅状态（借阅中、已归还、逾期、丢失）

### 6. 统计分析
- **图书统计**
  - 图书类别统计表
  - 各类别图书数量和可借数量
  - 热门图书排行

- **读者统计**
  - 活跃读者排行
  - 借阅次数统计

- **借阅统计**
  - 总借阅量
  - 当前在借数量
  - 逾期数量
  - 逾期记录明细

### 7. 系统设置
- **基本信息**
  - 查看系统版本
  - 当前用户信息

- **读者种类管理**
  - 新增种类
  - 设置借阅规则

## 🔍 典型使用场景

### 场景1：新读者注册
```
1. 进入"读者管理"
2. 点击"新增读者"
3. 填写信息：
   - 读者编号: 例如 S2024001
   - 姓名: 张三
   - 选择类别: 学生
   - 填写联系方式
4. 点击确定
5. 系统自动计算有效期（学生为365天）
```

### 场景2：采购新书
```
1. 进入"图书管理"
2. 点击"新增图书"
3. 填写信息：
   - ISBN: 978-7-111-12345-6
   - 书名: 数据结构与算法分析
   - 作者: 张三
   - 出版社: 机械工业出版社
   - 选择类别: 计算机科学
   - 数量: 5册
4. 点击确定
5. 图书录入成功，5册全部可借
```

### 场景3：完整借还流程
```
借书：
1. 读者来借书，提供读者证
2. 进入"借还管理"→"借书"
3. 扫描或输入读者编号: S2024001
4. 扫描或输入图书ISBN: 978-7-111-12345-6
5. 点击"确认借书"
6. 系统提示借书成功，显示归还日期

还书：
1. 读者来还书
2. 进入"借还管理"→"还书"
3. 搜索读者编号: S2024001
4. 找到该读者的借阅记录
5. 点击对应图书的"还书"按钮
6. 系统计算罚款（如有）
7. 确认还书成功
```

### 场景4：处理逾期图书
```
1. 进入"统计分析"→"借阅统计"
2. 查看逾期记录列表
3. 记录逾期读者信息
4. 联系读者催还
5. 读者还书时自动计算罚款
```

## ⚠️ 注意事项

1. **数据安全**
   - 定期备份数据库文件（位于用户数据目录）
   - 首次使用请修改默认管理员密码

2. **操作规范**
   - 借书前务必验证读者证有效性
   - 还书时检查图书损坏情况
   - 及时处理逾期记录

3. **系统维护**
   - 定期清理过期读者数据
   - 及时更新图书状态
   - 监控系统运行状态

## 🐛 常见问题

### Q1: 忘记管理员密码怎么办？
A: 需要删除数据库文件重新初始化系统，或联系技术支持。

### Q2: 如何批量导入图书数据？
A: 当前版本不支持批量导入，需逐本录入，后续版本将支持Excel导入。

### Q3: 读者编号规则是什么？
A: 建议使用统一格式，如：S+年份+序号（学生），T+年份+序号（教师）

### Q4: 图书丢失如何处理？
A: 在借阅记录中标记为"丢失"，系统会自动扣减库存并计算赔偿金。

### Q5: 如何查看某本书的借阅历史？
A: 在图书管理页面点击图书详情，或在统计分析中按图书ID查询。
